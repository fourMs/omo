Oslo Mobile Orchestra

Pick an instrument below. In each app use Learn and Audio on/off in the top-right corner.

Browsers: iPhone/iPad → Safari (Chrome/Firefox OK too). Android → Chrome (Firefox/Samsung Internet usually OK). Open the site in the browser — not inside Instagram, email, or other apps.

Permissions You need to enable the use of mic, camera and motion sensors when asked.

Volume: Should be turned up to at least 70%, possibly higher.

Install: Add to Home Screen for full-screen (works offline for core pages).

Device support — check mic, motion, and other APIs before a workshop.
