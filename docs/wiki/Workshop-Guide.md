## Goals

- Play together with phones only — no cables.
- Hear different roles: harmony, rhythm, gesture, timbre, AI.
- Optional: connect controls to synthesis and sensors.

## Before the room (15 min)

1. Test Wi‑Fi or hotspot; everyone uses the same site URL.
2. **iPhone → Safari**, **Android → Chrome**; avoid in-app browsers.
3. Conductor phone: full volume, Do Not Disturb.
4. Optional seating chart: row A = drones, B = rhythm, etc.
5. Demo: tap screen → **Audio on** → allow permissions.

## 45-minute script

### Warm-up (5 min)

Hub → **Synth Pad**. Everyone finds one note. Count 1–2–3–4, release on 4.

### Sensors (10 min)

**Motion Wah** or **Compass Wah** — tilt = timbre or volume. Conductor gestures; group follows without reading screens.

### AI (10 min)

**Train & Shake** — record still / sway / shake (3+ each). Open **Teach** panel: feature map, confusion matrix, compare k-NN vs tiny net.

**Evo Drumkit** (optional) — one player records a groove (4-beat count-in); group hears evolved variants.

Discuss: What confuses the model? Data stays on device until export.

### Ensemble (15 min)

1. **Conductor** — set BPM, **Start countdown**, share musician link via **QR** (or AirDrop).
2. **Drones** — 8× **Drone Choir** on different chord tones.
3. **Rhythm** — **Drumkit**, **Drum Sequencer Linear/Circular**, or **Firefly** in a circle (pulse sync).
4. **Harmony** — 4× **Harmonizer** listening to the room; hold to blend in.
5. **Texture** — **Granular Tilt** or **Video Sonifier** at the back.

### Cool-down (5 min)

One sustained **Drone Choir** tone. Discuss: musical vs “tech demo”?

## Accessibility

- Large touch targets; drones/rhythm need little precision.
- Relabel Train & Shake classes if needed (e.g. low / mid / high phone position).
- Visual readouts on sequencers and ML apps for HOH participants.

## Troubleshooting

| Problem | Fix |
|---------|-----|
| No sound | **Audio on**; tap screen first; iOS silent switch off |
| Mic/camera/motion blocked | Safari / Chrome; not in-app browser |
| Motion flat | Reload; iOS Settings → Safari → Motion |
| Out of sync | Conductor URL; expect ~50–100 ms without server |
| Harsh room | Lower volume; fewer noise gestures |
| Harmonizer feedback | Headphones or low speaker when holding tone |
| Firefly won’t lock | Louder neighbours; phones facing inward |

## Assessment prompts

2. Why is k-NN **supervised** learning?
3. Compare **Granular Tilt** vs **Drone Choir** for ambient vs harmonic roles.
4. How does **Firefly** entrain without a central server?
